# Spring Data JPA with Spring Boot and Hibernate

This folder contains the hands-on exercises completed as part of the **Cognizant Digital Nurture – Java Full Stack Engineer (Java FSE)** training program.

## Contents

### 1. Spring Data JPA - Quick Example
- Basic Spring Boot application with Spring Data JPA
- Hibernate configuration
- MySQL database connectivity
- Entity mapping
- Repository implementation
- Retrieve all countries from the database

### 2. Difference between JPA, Hibernate and Spring Data JPA
- Java Persistence API (JPA)
- Hibernate ORM Framework
- Spring Data JPA
- Comparison and relationship between JPA, Hibernate and Spring Data JPA

### 3. Hands-on 2 - Country Service
- Implement service layer for Country management
- Configure Spring Data JPA repository
- Service implementation using `JpaRepository`

### 4. Hands-on 3 - Find Country
- Retrieve a country using its country code
- Custom exception handling
- Service layer implementation

### 5. Hands-on 4 - Add Country
- Insert a new country into the database
- Save entity using Spring Data JPA
- Verify data insertion in MySQL
