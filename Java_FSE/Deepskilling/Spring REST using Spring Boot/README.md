# Spring REST Using Spring Boot 3

## Overview

This folder contains the hands-on exercises completed for the Spring REST using Spring Boot 3 module.

Each exercise is maintained in a separate project/folder for better organization and easy understanding.

---

## Technologies Used

- Java
- Spring Boot 3
- Spring Web
- Spring Security
- Spring Core
- Maven
- JWT (JJWT)
- Postman
- IntelliJ IDEA

## Learning Outcomes

- Create Spring Boot applications using Spring Initializr.
- Build RESTful web services.
- Configure Spring XML beans.
- Return Java objects as JSON responses.
- Use path variables in REST APIs.
- Configure Spring Security.
- Decode HTTP Basic Authentication credentials.
- Generate JWT tokens.
- Test REST APIs using Postman.
